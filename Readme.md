# Чат-приложение

Это простое чат-приложение, которое позволяет пользователям обмениваться сообщениями в реальном времени. Приложение поддерживает обычных пользователей и администраторов, которые могут удалять сообщения и блокировать пользователей.

## Установка

### 1. Клонируйте репозиторий

### 2. Установите зависимости

```bash
pip install -r req.txt
```

### 3. Установите ngrok

### 4. Запустите приложение 
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 5. Запустите ngrok, пробрасывая порт вашего WebSocket сервера (по умолчанию это порт 8080): 
```bash
ngrok http 8080
```
